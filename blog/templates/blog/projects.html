NYC Taxi Data Visualization

React • TypeScript • Django • DuckDB • Leaflet • Docker • Railway

An interactive web application that visualizes NYC taxi trip data using real-time processing of 60MB+ parquet files. Features an interactive map showing pickup locations, trip volumes, and fare analytics with data sourced directly from NYC's Taxi & Limousine Commission.

Key Features:

  • Interactive Leaflet map with clickable taxi pickup locations
  • Real-time DuckDB processing of 160K+ trips per month
  • Direct parquet file querying from NYC CloudFront CDN
  • Dynamic data visualization with color-coded fare ranges
  • Responsive React frontend with TypeScript
  • RESTful Django API serving analytics data
  • Docker containerization with Railway deployment
  • CORS-enabled single-port architecture
